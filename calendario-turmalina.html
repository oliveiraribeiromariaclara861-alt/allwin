O que é Turmalina

A turmalina é uma pedra de proteção, força e equilíbrio. Ela absorve o que pesa e devolve clareza. Quem carrega turmalina escolheu transformar. Esse é o nome dessa turma porque vocês não estão aqui só para indicar um produto. Estão aqui para crescer, aparecer e plantar algo que vai durar.

Consistência não pede que os resultados apareçam hoje. Pede que você plante mesmo assim. Cada vídeo, cada story, cada aparição é uma semente. O fruto vem para quem não para de plantar.
